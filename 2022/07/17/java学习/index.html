java
发布日期:   2022-07-17

闲聊

  • 阿里巴巴是国内java的权威
  • java的语法相对来说比c/c++简单
  • java刚开始叫oak,刚开始是做微波炉、智能家电这一块.
    • java的创始人喜欢喝咖啡，然后印度尼西亚有一个到叫爪哇岛，盛产咖啡.最后oak就变成了java.

java开发环境

  • java的编译器是JDK，公司不一定是用最新版的JDK,公司用的最多的版本是JDK1.7,JDK1.8
  • JRE是java的运行环境，就是玩我的世界需要下载的那个就是jre
  • JVM是java虚拟机
  • IDEA是java的集成开发环境

前三者的关系是JDK包着JRE,JRE包着JVM

HelloWorld

编程规范建议：左括号‘{’跟着当前行，右括号‘}’和当前行开头对齐.

  • public calss 是类
    • 一个java文件中只能有一个public的类
  • HelloWorld是类名
    • 类名需要和创建的java文件名相同
    • 类名用大驼峰的形式命名
  • 在public class HelloWorld里面的叫类体
  • 在public static void main(String[] args)里面的叫方法体

java编译过程

第一步，有一个.java后缀的文件，这文件里保存的是自己写的java代码

第二步，使用JDK的命令javac，javac空格+.java后缀的的文件全名包含后缀.
执行第二部后生成一个.clas后缀的文件，这个文件是字节码文件.

第三步，使用JDK命令java，java+文件名（生成的字节码文件的文件名和原来.java的文件名是一致的）

标识符

标识符可以理解为变量名，常量名、函数名等
标识符可以有英文、数字、下划线、$、组成

变量

java中没有全局变量
如果变量没初始化，则编译报错.

数据类型

包装类

Java中把基本数据类型包装成一个个独立的类

序号 数据类型 包装类 数据类型 所占大小单位/字节
1 byte Byte 字节型 1
2 char Character 字符型 2
3 short Short 短整型 2
4 int Integer 整形 4
5 long Long 长整形 8
6 float Float 单精度浮点型 4
7 double Double 双精度浮点型 8
8 boolean Boolean 布尔型 没有规定

整形int

在java中int类型没有无符号（unsigned），二进制第一个位始终代表符号位
int类型在32/64位下都是占4字节
int的取值范围是($-2^{63}$)~($2^{63}-1$)

长整型long

占8字节
long没有无符号（unsigned）,二进制第一个位始终代表符号位
long的取值范围是($-2^{63}$)~($2^{63}-1$)
创建long类型的变量时，要在初始值后面加上L，例如：long a = 10L;
如果不加L，那这个变量就是整形

短整型short

占2字节
取值范围：-32768~32767

双精度浮点型double

占8字节
和C语言的规则一样

单精度浮点数float

占4字节
用小数的时候建议用double
因为float只能表示到小数后6位，精度不如double高.

字符数据类型char

占2个字节
java中一个汉字占2字节
java中使用Unicode字符集.内容比ASCLL丰富
char的取值范围是0~65535

字节数据类型byte

占1个字节
取值范围-128~127

布尔类型boolean

在java中boolean大小没有明确规定占几个字节
布尔类型只有2个取值：true/false

在java中没有0是假，非0是真
只有true是真，false是假

拼接

拼接符号：“+”
在C语言中想输出中文加数字是printf("a+b的值是：%d",a+b);这种使用了%d形式输出叫格式化输出
在java中有一种新的输出方式：拼接.示例：System.out.println("a+b的值是："+ ( a + b ));
示例：

  • 任何字符串类型的数据和其他类型的数据进行拼接，最后输出的结果是字符串类型的.
  • java可以使用拼接输出，也可以使用格式化输出

整形提升

在表达式计算时，各种整型首先要提升为int类型，如果int类型不足以表示则要提升为unsigned int类型；然后执行表达式的运算。
java的整形提升规则和C语言一样

隐式类型转换

把大类型给小类型叫隐式类型转换
当不同数据类型进行计算的时候，需要将不同的数据类转换成同一种数据类型在进行计算.
精度低的数据类型向精度高的数据类型进行转换
具体转换的规则和C语言一样

显式类型转换

把小类型给打类型叫显式类型转换

final等同于const
