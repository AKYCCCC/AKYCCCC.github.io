java集合进阶

java学习
java
发布日期:   2022-12-03

拆装箱

在集合中，基本数据类型是不能作为数据类型直接用，要变成对应的包装类.

JDK5以前

jdk5以前使用基本数据类型作为集合的数据类型时，需要手动将元素先变成包装类然后在添加到集合中

JDK5以后

java为了方便做了优化，直接用包装类就可以.

底层还是拆箱装箱，只不过java自动帮你完成，不需要在自己手动拆装箱.

Integer i = 1;

``a.add(123); a.add(456);`

集合体系

集合有单列集合、双列集合

单列集合：一次添加一个元素

双列集合：一次添加一对元素（键值对）

单列集合

Collection

collection是所有==单列集合==的祖宗接口，它的功能可以被所有单列集合使用.

因为collection是一个接口，不能直接创建对象，所以使用多态创建collection的实现类对象

遍历collection实现类的三种通用方式

1.迭代器遍历（Iterator）

2.增强for

集合对象名.for()IDEA自动生成.

也可以手打

for(集合数据类型 自定义变量名 : 集合名) { 方法体; }

3.lambda表达式

使用对象名.forEach()方法，在参数中创建匿名类并重写匿名类方法.

然后对其进行简化

以上遍历方式均在各自的集合中演示.

因为List有索引，可以使用for循环遍历.而Set没有索引，不能使用for循环遍历.

而现在学的是collection，collection不能使用for循环.所以干脆不学for循环遍历，而是学习List和Set都能用的通用三种遍历方式

List

==List系列所有集合的特点：添加的元素有序（指入和取顺序）、元素可重复添加、有索引（可以使用for循环遍历）==

List中重载或者特有的方法.

ArrayList集合

ArrayList可以使用collection接口中的所有方法，也可以使用List接口中的所有方法

ArrayList底层是用动态数组实现.

优点是：读取效率高

缺点是：频繁的插入删除效率低.

基本数据类型操作
创建对象

多态创建：Collection<E> cl = new ArrayList<>();

直接创建：ArrayList<String> al = new ArrayList<>();

添加元素

al.add("aa")

al.add("bb")

al.add("cc")

没有知道添加的位置，就默认添加在末尾

添加成功返回true，反之false.

往List集合中添加元素时，只要数据类型符合要求那么add方法就返回true.因为List集合的元素可以重复.

清空元素

al.clear();

删除元素

根据给定的对象删除

al.remove("aa");

根据索引删除

al.remove(0);

因为collection定义的是共性的方法，所以不能通过索引删除，只能通过元素的对象删除.

删除成功返回true,反之false.

判断元素是否存在

al.contains();

存在true,反之false.

contains底层是通过数组的索引对数组中每一个元素用equals方法进行比较

判断集合是否为空

al.isEmpty();

空返回true,反之false

获取集合长度

al.size();

修改指定位置的元素

al.set(0,"dd");

修改成功后，返回值是原来的元素

获取索引位置的元素

al.get(0);

LinkedList集合

LinkedList集合可以使用collection接口中的所有方法.也可以使用List接口中的所有方法

LinkedList集合底层是双向链表实现的

优点是：增删效率高

缺点是：遍历效率低

以下是LinkedList集合特有的方法

特有的方法用的不多，更多是用collection接口和List接口中的方法.

LinkedList底层实现

底层就是标椎的双向链表

有一个节点Node

Node中有三个属性，一个记录上一下节点的地址，一个记录元素值，一个记录下一个节点的地址

然后有一个LinkedList类

类中的成员属性记录头节点和尾节点的地址
